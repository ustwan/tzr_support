1️⃣ КЛОНИРУЙ РЕПОЗИТОРИЙ
═══════════════════════════════════════════════════════════════════════

git clone https://github.com/ustwan/tzr_support.git
cd tzr_support


2️⃣ НАСТРОЙ .env
═══════════════════════════════════════════════════════════════════════

cp .env.example .env
nano .env

Обнови:

# Telegram (ОБЯЗАТЕЛЬНО)
TELEGRAM_BOT_TOKEN=8499545269:AAEXWsNh-YfrncsCiZP31x4V3wDwgdgHHOM
FEEDBACK_TELEGRAM_CHANNEL_ID=-1003105374647

# WebSocket (получи JWT на сайте)
SITE_WS_URL=wss://tzreloaded.ru/ws/feedback-bot?token=<JWT>
HMAC_SECRET=<тот же что на сайте>

Сохрани: Ctrl+O, Enter, Ctrl+X


3️⃣ ЗАПУСТИ DOCKER
═══════════════════════════════════════════════════════════════════════

docker-compose up -d


4️⃣ ПРОВЕРЬ ЛОГИ
═══════════════════════════════════════════════════════════════════════

docker-compose logs -f

Должно быть:
✅ WebSocket connected to site!
✅ Telegram bot handlers registered


5️⃣ ПРОТЕСТИРУЙ
═══════════════════════════════════════════════════════════════════════

В Telegram:
• Напиши боту @tzr_support_bot: /start
• Создай обращение: /feedback

Результат: Обращение появится в группе с ticket_tg_1


═══════════════════════════════════════════════════════════════════════

✅ ГОТОВО! БОТ РАБОТАЕТ!

Остановка: docker-compose down
Перезапуск: docker-compose restart
Логи: docker-compose logs -f feedback-bot
